Demo: Inline ICE Code Editor via Polymer

The ICE Code Editor, which is used throughout 3D Game Programming for Kids, can also be used on web pages to illustrate progressing concepts. The two live code editors below are added to the document with <ice-code-editor> tags, thanks to the power of Polymer.
